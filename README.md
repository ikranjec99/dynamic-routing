# dynamic-routing

## Description

Purpose of this project is to demonstrate beginner friendly implementation of dynamic routing.

## Getting Started

### Dependencies

* React Suite - v5.15.1
* React Router - v6

### Installing

* Clone project

* Install node modules
```
npm install
```

### Executing program

* Navigate to project root folder 
  
* Run project
```
npm start
```
